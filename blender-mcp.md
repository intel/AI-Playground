# Usage with official Blender MCP

## Prerequisites

In a folder of your choice (e.g. `C:\Users\username\src\blender\`):

```
git clone https://projects.blender.org/lab/blender_mcp.git
```

## Setup Plugin in Blender

This needs Blender `5.1.0` released on March 17, 2026. The `blender_manifest.json` specifies that.

1. Go into that folder, navigate `./addons/`, right click `blender_mcp_addon/` folder and "Compress to zip file"
2. In Blender, go to Edit Menu > Preferences... > Add-ons > Install from disk... and select the previously created `blender_mcp_addon.zip` file (folder won't work).

A new entry 'MCP' appears in the list. Expand it, click "Start MCP Bridge Server".

You might need to enable online access under Preferences... > System > Network > Allow Online Access.


## Setup the MCP server in your agentic chat client

in the cloned repo root, find the path:
- setup a venv
- activate the venv
- `pip install .`
- `Get-Command blender-mcp` to find the path to the executable, which should env with `...venv\Scripts\blender-mcp.exe` on Windows, e.g.

C:\Users\username\src\blender\blender_mcp\venv\Scripts\blender-mcp.exe

Configure Ai Playground (as a developer):

in mcpServers.ts, extend the existing `mcpServers` object with a new entry, e.g.

```
export const mcpServers: Record<string, McpServerConfig> = {
  blender: {
    id: 'blender',
    name: 'Blender MCP (official)',
    command: 'C:\\Users\\username\\src\\blender\\blender_mcp\\venv\\Scripts\\blender-mcp.exe',
    args: [],
  },
  ```

# Usage with the Ahujasid Blender MCP

https://github.com/ahujasid/blender-mcp

## Setup Blender

install the blender addon

https://github.com/ahujasid/blender-mcp?tab=readme-ov-file#installing-the-blender-addon

which is roughly:
- download plugin.py from github
- in Blender, go to Edit Menu > Preferences... > Add-ons > Install from disk...

start the mcp server
- on the right edge, there is a tiny left arrow right next to the red blue green xyz axis, click it
- there appears a 90 deg rotated text "Blender MCP", click it
- click "Connect to MCP Server"

## Setup AIPG

- Set Chat mode
- Set Agentic preset
- Ensure "mcp/blender" is green in settings sidebar left

## Usage

Ask to create a pyramid with tip down, and place it on top of the cube.

Ask to add a second cube, rotate it by 45 degrees, and change its color to red.

## Reference: available tools

Here is the list of available tools:

- `comfyUI` - Generates images based on text prompts with customizable workflows, variants, and parameters.
- `comfyUiImageEdit` - Edits existing images using various workflows like Edit By Prompt, Upscale, Colorize, and more.
- `visualizeObjectDetections` - Visualizes object detections on an image by drawing bounding boxes and labels.
- `blender__get_scene_info` - Retrieves detailed information about the current Blender scene.
- `blender__get_object_info` - Retrieves detailed information about a specific object in the Blender scene.
- `blender__get_viewport_screenshot` - Captures a screenshot of the current Blender 3D viewport.
- `blender__execute_blender_code` - Executes arbitrary Python code in Blender.
- `blender__get_polyhaven_categories` - Retrieves categories for a specific asset type on Polyhaven.
- `blender__search_polyhaven_assets` - Searches for assets on Polyhaven with optional filtering.
- `blender__download_polyhaven_asset` - Downloads and imports a Polyhaven asset into Blender.
- `blender__set_texture` - Applies a downloaded Polyhaven texture to an object in Blender.
- `blender__get_polyhaven_status` - Checks if PolyHaven integration is enabled in Blender.
- `blender__get_hyper3d_status` - Checks if Hyper3D Rodin integration is enabled in Blender.
- `blender__get_sketchfab_status` - Checks if Sketchfab integration is enabled in Blender.
- `blender__search_sketchfab_models` - Searches for models on Sketchfab with optional filtering.
- `blender__get_sketchfab_model_preview` - Gets a preview thumbnail of a Sketchfab model by its UID.
- `blender__download_sketchfab_model` - Downloads and imports a Sketchfab model by its UID.
- `blender__generate_hyper3d_model_via_text` - Generates a 3D asset using Hyper3D from a text description.
- `blender__generate_hyper3d_model_via_images` - Generates a 3D asset using Hyper3D from input images.
- `blender__poll_rodin_job_status` - Polls the status of a Hyper3D Rodin generation task.
- `blender__import_generated_asset` - Imports the asset generated by Hyper3D Rodin after the task completes.
- `blender__get_hunyuan3d_status` - Checks if Hunyuan3D integration is enabled in Blender.
- `blender__generate_hunyuan3d_model` - Generates a 3D asset using Hunyuan3D from text or image input.
- `blender__poll_hunyuan_job_status` - Polls the status of a Hunyuan3D generation task.
- `blender__import_generated_asset_hunyuan` - Imports the asset generated by Hunyuan3D after the task completes.

These tools can be used for image generation, editing, 3D modeling, asset downloading, and scene management in Blender and related workflows.

The important ones are:

- `blender__execute_blender_code`
- `blender__get_object_info`
- `blender__get_scene_info`
- `blender__get_viewport_screenshot`
